1. Tentang Pemrograman Internet

WEB adalah proses pembuatan aplikasi web menggunakan bahasa pemrograman dan teknologi web. Ini meliputi pembuatan halaman web statis seperti HTML dan CSS, serta halaman web dinamis yang menggunakan bahasa pemrograman seperti JavaScript, PHP, dan Ruby. Pemrograman web mencakup perencanaan, konseptualisasi, pemodelan pengiriman media elektronik melalui internet. Proses pembuatan program dibantu dengan software khusus, dapat berupa IDE (Integrated Development Environment) dalam bentuk command line interface maupun GUI (Graphical User Interface). Orang yang mengembangkan aplikasi website disebut dengan web developer. Pemrograman web adalah proses mengkomunikasikan perintah pada komputer untuk membangun suatu program berbasis website agar dapat diakses menggunakan internet. Program atau situs web yang terbentuk dari pemrograman web baru akan bisa memberikan informasi dari halaman browser yang diakses menggunakan koneksi internet.


2. Mengapa kita harus belajar pemrograman internet

Ada beberapa alasan mengapa seseorang harus melakukan pemrograman internet, diantaranya:

  • Menunjang Karir: web programming adalah hal yang banyak dicari saat ini, dan diprediksi akan terus semakin dibutuhkan. Diproyeksikan sekitar 21,800 lowongan kerja dibuka, rata-rata, setiap tahunnya dalam satu dekade.

  • Melatih Ketelitian: bahasa yang digunakan untuk web programming memiliki sintaks dan aturan penulisan kode masing-masing. Nah, agar program yang Anda buat dapat berjalan, maka Anda harus teliti menerapkan sintaks tersebut. Dengan begitu, ketika Anda terbiasa menulis kode, maka pada saat bersamaan Anda juga sedang melatih ketelitian.
